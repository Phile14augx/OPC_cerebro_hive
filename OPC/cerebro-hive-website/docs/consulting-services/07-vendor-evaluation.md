# 2.7 AI Vendor Evaluation & Selection

**What it is:** An objective, structured process to evaluate and select AI vendors, platforms, and tools — without the bias of vendor kickbacks or preferred partnerships. CerebroHive acts as a neutral technical advisor.

**What we evaluate:** LLM providers, AI platforms (Azure AI, Google Vertex AI, AWS Bedrock, Cohere, Anthropic), automation tools (UiPath, Automation Anywhere, Microsoft Power Platform), data platforms, MLOps tools, vector databases, AI application builders.

**Deliverables:**
- Requirements Definition Document — functional, technical, compliance, and cost requirements
- Market Scan Report — long-list of 15–30 potential vendors per category
- Vendor Scorecard (weighted evaluation matrix)
- RFP/RFI Template and management support
- Vendor Demo Facilitation (structured demonstrations with consistent scoring)
- Total Cost of Ownership (TCO) model for top 3 finalists
- Final Selection Recommendation Report with risk assessment
- Contract Review Guidance (commercial terms, SLAs, exit clauses)

**Pricing:** ₹3,00,000 – ₹12,00,000 / $4,000 – $15,000
