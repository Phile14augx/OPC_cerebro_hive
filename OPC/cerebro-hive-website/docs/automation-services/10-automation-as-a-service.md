# 3.10 Automation-as-a-Service (Managed Automation)

**What it is:** CerebroHive takes full operational responsibility for a client's automation infrastructure — monitoring, maintaining, iterating, and expanding automations on an ongoing monthly retainer basis.

**Included in the managed service:**

- 99.5% uptime SLA on all deployed automations
- 24-hour response time for automation failures
- Monthly performance reports (automations run, errors, time saved, cost avoided)
- Ongoing iteration — refining automations based on business changes
- New automation additions within included hours
- Quarterly automation portfolio review and expansion recommendations
- Dedicated account manager and technical contact
