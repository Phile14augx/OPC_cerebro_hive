# 3.9 Data Pipeline & Analytics Automation

**What it is:** Construction of automated data pipelines that extract data from multiple sources, transform and clean it, load it into analytics platforms, and trigger automated reporting and alerting — eliminating manual data wrangling.

**What we build:**

- **ETL/ELT pipelines:** From source systems (CRM, ERP, databases, APIs, flat files) to data warehouse (BigQuery, Snowflake, Redshift, PostgreSQL)
- **Real-time data streaming:** Event-driven pipelines for operational data that needs near-real-time availability (Kafka, AWS Kinesis, Pub/Sub)
- **Automated reporting:** Scheduled dashboards and reports auto-generated and distributed (email, Slack, Notion, PDF)
- **Anomaly detection pipelines:** Statistical models that monitor KPIs and alert when values deviate from expected ranges
- **AI-powered data quality:** Automated checks that flag data quality issues and route them for remediation
- **Revenue intelligence automation:** Pipeline health scores, churn signals, and growth opportunity alerts surfaced automatically to sales leadership

**Tools:** dbt, Airbyte, Apache Airflow, Fivetran, Metabase, Looker, Power BI, Tableau, Google Data Studio.
