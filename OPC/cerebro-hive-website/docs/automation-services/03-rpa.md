# 3.3 Robotic Process Automation (RPA)

**What it is:** Implementation of software robots that mimic human interactions with desktop and web applications to automate high-volume, rule-based processes — primarily for legacy system integration where APIs don't exist.

**Platforms:** UiPath, Automation Anywhere, Microsoft Power Automate Desktop, Blue Prism.

**Key RPA use cases:**

- **Finance & Accounting:** Bank reconciliation, accounts payable/receivable data entry, financial close processes, audit data extraction, expense report processing
- **Human Resources:** Payroll processing, employee data entry and maintenance, leave management, compliance reporting
- **Supply Chain & Procurement:** Purchase order processing, invoice matching (3-way match), vendor data management, inventory updates
- **Banking & Insurance:** KYC document verification, loan application processing, claims data entry, regulatory report generation
- **Healthcare:** Patient record updates, insurance claims submission, appointment scheduling, billing code extraction
- **Government & Compliance:** Form processing, regulatory filing, data migration between legacy systems

**Engagement includes:**
- Process selection and ROI analysis (not all processes are good RPA candidates)
- Process design document (PDD) — standardized process documentation for bot design
- Bot development and testing (unit, integration, UAT)
- Control room configuration and bot scheduling
- Exception handling design
- Go-live support and hypercare period
- Ongoing monitoring and maintenance

**Pricing:** ₹3,00,000 – ₹20,00,000 / $4,000 – $25,000 per bot/process (volume pricing for large portfolios)
