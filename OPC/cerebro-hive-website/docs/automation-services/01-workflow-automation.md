# 3.1 Workflow Automation (No-Code / Low-Code)

**What it is:** Design and implementation of automated workflows using leading no-code and low-code automation platforms — connecting existing business tools without custom software development.

**Platforms we build on:** n8n, Make (Integromat), Zapier, Microsoft Power Automate, Airtable Automations, Notion Automations, HubSpot Workflows, Salesforce Flow.

**Common use cases we automate:**

- **Lead management:** CRM entry from form submissions, lead scoring, assignment routing, follow-up sequence triggering
- **Invoice & billing:** Invoice generation from project milestones, payment reminder sequences, accounts receivable alerts
- **Onboarding workflows:** New employee/client onboarding — account provisioning, document collection, welcome sequences, task assignment
- **Reporting automation:** Data pulls from multiple sources → formatted report → auto-distributed to stakeholders on schedule
- **Social media scheduling:** Content approval → platform publishing → performance data pull → consolidated analytics report
- **Support ticket routing:** Inbound ticket classification, priority assignment, team routing, SLA monitoring, escalation triggers
- **Document workflows:** Form submission → document generation → e-signature → filing → CRM update

**Deliverables per engagement:**
- Process mapping and automation blueprint
- Platform selection recommendation (if not already using one)
- Workflow build and testing
- Documentation and runbook
- Training for client team
- 30-day post-launch support

**Pricing:** ₹50,000 – ₹5,00,000 / $700 – $7,000 per workflow package (complexity-dependent)
