# 5.3 Open Source Projects

**CerebroHive OSS Portfolio (planned/in development):**

**AgentKit-India**
A library of pre-built AI agent templates optimized for Indian business contexts — including WhatsApp-first interfaces, regional language support (Hindi, Tamil, Telugu, Kannada, Marathi), Indian regulatory compliance helpers (GST, TDS, PF), and integration connectors for popular Indian business tools (Tally, Zoho, Razorpay, Cashfree, IndiaMART).

**PromptVault**
An open-source prompt library and management system. Allows teams to store, version, test, and share prompts as code — with evaluation tooling to measure prompt performance across model updates.

**FlowGraph**
An open-source visual workflow designer for AI agent pipelines — drag-and-drop interface for designing LangGraph-compatible multi-agent workflows without writing orchestration code.

**AuditAI**
An open-source AI governance toolkit — model cards generator, bias testing utilities, audit log formatter, and compliance checklist runner. Designed to help developers add governance artifacts to their AI systems with minimal friction.
