# 4.1 AI Foundations Course

**What it is:** The entry point to CerebroHive Academy — a self-paced, beginner-friendly course that teaches non-technical professionals and business users how AI works, what it can do, and how to start using it effectively in their work.

**Target audience:** Business professionals, managers, entrepreneurs, and career changers with no prior technical background.

**Curriculum (8 modules, ~12 hours total):**

| Module | Topics |
|---|---|
| 1. What is AI? | History, key terminology, types of AI, current capabilities and limits |
| 2. How LLMs Work | Transformers explained simply, training, fine-tuning, tokens, context windows |
| 3. Prompt Engineering Fundamentals | Prompt structure, zero-shot/few-shot, chain-of-thought, system prompts |
| 4. AI Tools Landscape | ChatGPT, Claude, Gemini, Copilot — how to choose and when to use each |
| 5. AI in Business Functions | Marketing, finance, HR, operations, customer support — real-world use cases |
| 6. AI Risks & Limitations | Hallucinations, bias, privacy, IP — what professionals must know |
| 7. Automating Your Work with AI | No-code tools, Zapier, Make, ChatGPT plugins, workflow design |
| 8. Building Your AI Workflow | Personalized AI productivity system — frameworks and implementation |

**Format:** Video lessons + downloadable templates + quizzes + community access
