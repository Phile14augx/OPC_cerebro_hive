# 4.4 AI Automation Bootcamp

**What it is:** An intensive, hands-on program that takes participants from zero to deploying real automations and AI agents — practical skill-building focused on tools used in actual business environments.

**Format:** 8-week live cohort (2 × 2-hour sessions per week) + weekly labs + final project

**Curriculum:**

| Week | Topic | Tools Used |
|---|---|---|
| 1 | Automation Fundamentals & Mindset | Conceptual foundations, process mapping |
| 2 | No-Code Automation | n8n, Make, Zapier — building first workflows |
| 3 | API Integration | REST APIs, webhooks, authentication, Postman |
| 4 | AI-Powered Workflows | OpenAI API + n8n, ChatGPT in workflows |
| 5 | Building AI Agents | LangChain fundamentals, agent patterns, tool use |
| 6 | Data Automation | Airtable, Google Sheets automation, data pipelines |
| 7 | Production & Deployment | Error handling, logging, monitoring, maintenance |
| 8 | Final Project Presentations | Deploy a real automation for a real problem |

**Final project:** Each participant designs and deploys an original automation for their own work or business — reviewed and certified by CerebroHive instructors.

**Prerequisites:** Basic computer literacy; no coding experience required (coding-optional track available for developers who want to go deeper).

**Outcome:** CerebroHive Automation Practitioner Certificate + portfolio project

**Pricing:** ₹29,999 / $400 per participant; early cohort discount: ₹19,999; corporate batch: ₹2,00,000 for up to 10
